## Описание сред

### tesolo_delto_env

Просто кисть Tesolo Delto

### tesolo_delto_UR_env 

Среда с кистью на манипуляторе. В наблюдении данные о координатах объекта и пальцев

### tesolo_delto_UR_camera_env 

Среда с кистью на манипуляторе и камерой. Упрощенное наблюдение + кадр. Немного упрощенная награда (в сравнении со средой без камеры)
